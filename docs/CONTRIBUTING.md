# Contribuindo para Aurora Sovereign Intelligence

Obrigado por considerar contribuir para a Aurora! Suas contribuições são essenciais para o sucesso deste projeto.

## Diretrizes de Contribuição

### Antes de Começar

1. Faça um fork do repositório
2. Clone seu fork localmente
3. Crie uma branch para sua feature

### Processo de Contribuição

1. **Código:** Escreva código limpo e bem documentado
2. **Testes:** Inclua testes para novas funcionalidades
3. **Documentação:** Atualize a documentação conforme necessário
4. **Commit:** Use mensagens de commit claras e descritivas
5. **Pull Request:** Descreva suas mudanças em detalhes

### Padrões de Código

- Siga PEP 8
- Use type hints
- Escreva docstrings em português
- Máximo 100 caracteres por linha

### Testes

Todas as contribuições devem incluir testes:

```bash
python -m unittest discover tests
```

## Reportando Bugs

Use GitHub Issues para relatar bugs. Inclua:

- Descrição clara do bug
- Passos para reproduzir
- Comportamento esperado vs. observado
- Ambiente (SO, Python version, etc)

## Sugestões de Melhorias

Sugestões são bem-vindas! Crie um GitHub Issue descrevendo:

- Problema que a melhoria resolve
- Solução proposta
- Exemplos de uso
